| Flash memory               | Winbond W25Q256JVFQ                                  |
| Network                    | Local network wired connection                       |
| Storage                    | Kingston KC3000 512GB M.2 2280 PCIe Gen4 x4 NVMe SSD |
| Power Supply               | Seasonic VERTEX GX-750 (750W, 80 Plus Gold)          |
